# DevSynthetix First Contribution

This repository is designed to help beginners make their
**first open-source contribution** in a structured and meaningful way.

It focuses on simple, non-code changes such as documentation,
naming improvements, and clarity fixes.

## Who is this for?
- Beginners new to GitHub
- First-time contributors
- Anyone learning pull requests

## How to contribute
1. Fork this repository
2. Pick an issue labeled `good first issue`
3. Make the requested change
4. Open a pull request

No prior experience is required.

## What you will learn
- How pull requests work
- How to follow contribution instructions
- How open-source collaboration happens

This repository is actively maintained.
